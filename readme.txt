== Installation ==

1. Upload `hulvire-slider` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place in your pages
`[huu_slider]`
1. in themes 
`huu_get_slider();`
